# Network Monitor

**Network Monitor** — це десктопний кросплатформлений додаток створений для розуміння мереж(ICMP пакети, та TCP ),перевірки TCP портів та пінгу на віддалених серверах, усі перевірки будуть зберігатись у ізольованій базі даних, також програма виконує CRUD  операції та інших зручних для користувача функцій

## Основні можливості 

- **Два режими сканування:** 
  - `Ping` — перевірка загальної доступності хоста.
  - `Port` — перевірка відкритого порту на вузлі.
- **База даних:** Усі результати автоматично зберігаються в локальну БД mysql.
- **Експорт даних:** Можливість вивантажити всю історію сканувань у файл формату `.csv` для подальшого аналізу та розрахунків вже локально у табличній формі.
- **Двомовний інтерфейс:** Користувач може вибрати якою мовою буде відображатись gui.
- **Керування даними:** 
  - Доступні усі CRUD операції.
  - Видалення окремих записів через контекстне меню (правий клік по рядку в таблиці).
- **Гаряча клавіша:** - при записі у вікні нажати <ENTER>
## Технології та Архітектура

Проект у більшості має вбудовані для Pathon бібліотеки, якщо є інтегровані - то можна подивитись у <requirements.txt>

- **Мова:** Python 
- **GUI:** tkinter
- **База даних:** mysql
- **Мережа:** `socket`, `ping3`
- **Архітектура:** MVC для чіткого розділення логіки, інтерфейсу та роботи з даними.
- **Віртуалізація:** Docker-compose

## Встановлення та Запуск

 Щоб запустити легко програму слід виконати такі кроки:

Запустити базу даних 
 - ``` cd docker
 - ``` docker compose up -d
Завантажити програму із залежностями і віртуальним середовищем
 - ``` cd ..
 - ``` chmod +x setup.sh
 - ``` ./setup.sh
Запустити програму
 - ``` chmod +x run.sh
 - ``` ./run.sh

Або:
- ```python3 -m venv venv
- ```source venv/bin/activate

Також можна у терміналі:
- ```python3 src/main.py
